On this page you can customize all data related to Monsta configuration and also restore database backups.

![image-1756128994828.png](../../../../../assets/images/p60_image-1756128994828.png)


| Option | Description |
| :---: | :--- |
| ![image-1647434370176.png](../../../../../assets/images/p60_image-1647434370176.png) | When enabled, inverts Monsta's screen colors. |
| ![image-1647435194179.png](../../../../../assets/images/p60_image-1647435194179.png) | Allows creating, editing, and removing existing templates and monitors. For more information see: [Templates](/en/manual/configuracoes/templates). |
| ![image-1647435290317.png](../../../../../assets/images/p60_image-1647435290317.png) | Allows customizing agent settings and blocking them. For more information see: [Agents](/en/manual/configuracoes/agentes). |
| ![image-1647435557586.png](../../../../../assets/images/p60_image-1647435557586.png) | Allows adding and removing device groups as well as managing them. For more information see: [Device Group](/en/manual/configuracoes/grupo-de-dispositivos). |
| ![image-1647435611989.png](../../../../../assets/images/p60_image-1647435611989.png) | Allows creating, editing, and removing data sources, instance searches, uptime methods and actions. For more information see: [Monsta Studio](/en/manual/configuracoes/monsta-studio). |
| ![image-1647435653352.png](../../../../../assets/images/p60_image-1647435653352.png) | Manages Monsta's image library. For more information see: [Images](/en/manual/configuracoes/imagens). |
| ![image-1739982775972.png](../../../../../assets/images/p60_image-1739982775972.png) | This is Monsta's sound library where files that will be available for alerts are managed. For more information see: [Sounds](/en/manual/configuracoes/sons). |
| ![image-1647435745876.png](../../../../../assets/images/p60_image-1647435745876.png) | Switches between Monsta's update channels. The following channels can be chosen:<br />- **Official**: Channel with updates considered stable.<br />- **Beta**: Channel with updates in the testing phase.
| ![image-1647436547485.png](../../../../../assets/images/p60_image-1647436547485.png) | Manages users in Monsta. For more information see: [Users](/en/manual/configuracoes/usuarios). |
| ![image-1647436903151.png](../../../../../assets/images/p60_image-1647436903151.png) | Allows changing Monsta's general configuration parameters.<br /><aside class="starlight-aside starlight-aside--danger"><p class="starlight-aside__title">Warning</p>Incorrectly configured parameters can render Monsta inoperable.</aside><br />For more information see: [Parameters](/en/manual/configuracoes/parametros). |
| ![image-1647437280055.png](../../../../../assets/images/p60_image-1647437280055.png) | Allows enabling or disabling features available in Monsta. For more information see: [Plugins](/en/manual/configuracoes/plugins). |
| ![image-1756129051901.png](../../../../../assets/images/p60_image-1756129051901.png) | In this option you can view the status of Monsta's services and, if necessary, restart them individually. For more information see: [Services](/en/manual/configuracoes/servicos). |
| ![image-1647435477277.png](../../../../../assets/images/p60_image-1647435477277.png) | In this option you can restore a backup of Monsta's configurations. For more information see: [Cloud Backup](/en/manual/configuracoes/backup-em-nuvem). |
| ![image-1739983055127.png](../../../../../assets/images/p60_image-1739983055127.png) | Install and manage your certificates to access Monsta's web interface. It is possible to use your own certificates, certificates generated by Monsta, or automate them through Letsencrypt. For more information see: [TLS Certificate](/en/manual/configuracoes/certificado-tls). |
| ![image-1647437341171.png](../../../../../assets/images/p60_image-1647437341171.png) | Displays information about the license key in use. For more information see: [License Key](/en/manual/configuracoes/chave-de-licenca). |
| ![image-1647437392464.png](../../../../../assets/images/p60_image-1647437392464.png) | Displays information about the current version. |
