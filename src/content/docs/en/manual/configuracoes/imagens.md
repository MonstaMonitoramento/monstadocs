Manage your images easily: upload, organize by groups and custom names, and delete what you no longer need. Your image library, your way.

![image-1739990239560.png](../../../../../assets/images/p74_image-1739990239560.png)

| Ícone / Opção | Descrição |
| :---: | :--- |
| Nome | Sets a name for the image. |
| Grupo | Specifies which group the image belongs to. |
| ![image-1739990383549.png](../../../../../assets/images/p74_image-1739990383549.png) | Displays the identifier of the selected image. |
| ![image-1739990449233.png](../../../../../assets/images/p74_image-1739990449233.png) | Removes the selected image |
| ![image-1739990471015.png](../../../../../assets/images/p74_image-1739990471015.png) | Saves the selected image and its attributes. |
| ![image-1739990495811.png](../../../../../assets/images/p74_image-1739990495811.png) | Uploads a file to the image library. |
| ![image-1739990524318.png](../../../../../assets/images/p74_image-1739990524318.png) | Filters the image list according to the entered text. |
| ![image-1739990595935.png](../../../../../assets/images/p74_image-1739990595935.png) | Displays only the images belonging to the selected group. |
