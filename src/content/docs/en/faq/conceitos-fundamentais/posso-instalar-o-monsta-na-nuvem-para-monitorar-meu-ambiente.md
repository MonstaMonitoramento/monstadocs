---
title: "Can I install Monsta in the cloud to monitor my environment?"
---

Yes, Monsta is fully compatible with cloud environments. You can install it on virtual server instances provided by cloud providers such as AWS, Azure, Google Cloud Platform and others. This offers flexibility, scalability and reliability for your Monsta installation.

Our data collection technology and intelligent caching operate very well over WANs, reducing hardware and memory usage as well as minimizing internet data usage.

To monitor your network without the need for a VPN, use the Monsta [Agent](/en/start/instalacao/agente-instalacao-zero-conf).
