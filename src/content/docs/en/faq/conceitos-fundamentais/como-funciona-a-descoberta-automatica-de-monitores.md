---
title: "How does automatic monitor discovery work?"
---

Automatic discovery of monitors in Monsta is a feature designed to simplify and speed up the process of adding monitors to your device. Instead of adding each item manually, Monsta uses different protocols and methods to automatically identify the elements present on your equipment, adding the relevant monitors for each rule created. New monitors are created as they appear, and they will be removed if they cease to exist, for example, the creation or removal of a VPN interface on your device.

**How to create automatic monitor rules?**

- Click on the device for which you want to create the rule;
- In the bottom tab, click the icon: ![image-1746721039162.png](../../../../../assets/images/p98_image-1746721039162.png)
- In the window that appears, click to add a new rule;
- Select the monitor you want to be created automatically.
- You can define parameters for each rule. For more information, see [Automatic Monitors](/en/manual/dispositivos/monitores-automaticos).
