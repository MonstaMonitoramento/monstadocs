---
title: "What features does Monsta offer?"
---

Monsta offers a robust set of features for comprehensive IT infrastructure monitoring. These features are designed to provide visibility, proactive problem detection, efficient alerts, and analysis tools to ensure the health and performance of your systems and services.

## Monsta's Key Features

### 1. Real-time Monitoring

- **Server Monitoring**: Tracking metrics such as CPU, RAM, disk (free space, IOPS), network utilization, running processes and services (Windows, Linux).
- **Network Monitoring**: Monitoring network devices (routers, switches, firewalls) via SNMP, ICMP and other protocols, tracking traffic, interface utilization, errors, status and routing tables.
- **Application and Services Monitoring**: Monitoring web applications (HTTP, HTTPS), mail servers (SMTP, POP3, IMAP), DNS and other critical services, checking availability and performance.
- **Virtual Environment Monitoring**: Support for monitoring virtualization platforms such as VMware and Hyper-V, tracking the performance of hosts and virtual machines.

### 2. Visualization and Dashboards

- **Customizable Dashboards**: Creation of intuitive, dynamic graphical dashboards to display the most important metrics according to the needs of each user or team.
- **Real-time Charts**: Graphical visualization of resource performance over time, making it easier to identify trends and anomalies.
- **Infrastructure Maps**: Visual representation of the network topology and relationships between monitored devices and services.
- **Custom Views**: Creation of specific views for different areas of the infrastructure or for users with different access levels.

### 3. Alerts and Notifications

- **Threshold Configuration**: Defining limits for monitored metrics, generating alerts when those limits are exceeded.
- **Notifications**: Sending alerts via email, SMS and Telegram.

### 4. Analysis and Reporting

- **Historical Data**: Storage of monitoring data for long-term trend analysis and reporting purposes.
- **Root Cause Analysis**: Tools to help identify the root cause of performance issues or failures.
- **Capacity Planning**: Historical data can be used to forecast future capacity needs.

### 5. Other Important Features

- **Automatic Cloud Backup**: Securing configuration and monitoring data.
- **User and Permission Management**: Access control to the platform with different permission levels.
- **API (Application Programming Interface)**: Ability to integrate Monsta with other tools and systems.
- **Agents and Remote Monitoring**: Flexibility to monitor resources in different locations.
- **Ease of Use**: Intuitive interface and simplified configuration.
