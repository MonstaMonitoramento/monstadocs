To download Monsta, follow these steps:

- **Access the download site**: Go to [https://monsta.com.br/downloads](https://monsta.com.br/downloads).
- **Choose the Operating System**: Monsta can be downloaded for Linux distributions that use .RPM packages, such as Fedora and Red Hat, as well as for those that use .DEB, such as Debian and Ubuntu.
- **Follow the installation instructions**: See the [Installation Manual](/en/start/instalacao/instalacao-monsta#instalação) for more information on how to install Monsta on your server.
