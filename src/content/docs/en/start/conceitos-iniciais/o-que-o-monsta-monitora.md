Monsta is a comprehensive IT monitoring solution that provides visibility into various aspects of your infrastructure. It can monitor:

**Physical Infrastructure**:

- **Servers**: Performance (CPU, memory, disk, network), processes, services, hardware, etc.
- **Network**: Routers, switches, firewalls, Wi‑Fi and OLTs.
- **Power**: Uninterruptible Power Supplies (UPS) and generators.
- **Others**: Printers, IoT devices.

**Operating Systems**:

- **Windows**: Services, performance counters, users, processes, etc.
- **Linux**: Processes, resource usage, services, users, etc.

**Applications and Services**:

- **Web**: Availability (HTTP/HTTPS), response time, content, SSL certificates.
- **Email**: SMTP, POP3, IMAP (availability).
- **DNS**: Availability and performance.
- **Custom**: Via scripts or APIs.

In summary, Monsta monitors the performance and availability of **servers, network, applications, operating systems and cloud services** via protocols and agents. This ensures a complete view of the health of your IT infrastructure.
