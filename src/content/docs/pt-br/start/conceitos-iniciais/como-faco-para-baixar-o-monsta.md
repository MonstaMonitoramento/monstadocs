---
title: "Como faço para baixar o Monsta?"
sidebar:
  order: 5
---

Para baixar o Monsta, siga estes passos:

- **Acesse o site para download**: Vá para [https://monsta.com.br/downloads](https://monsta.com.br/downloads).
- **Escolha o Sistema Operacional**: O Monsta pode ser baixado para distribuições Linux que utilizam pacotes .RPM, como Fedora e Red Hat, como também para aquelas que utilizam .DEB, como Debian e Ubuntu.
- **Siga as instruções de instalação**: Acesse o [Manual de Instalação](/pt-br/start/instalacao/instalacao-monsta#instalação) para maiores informações sobre como instalar o Monsta em seu servidor.
