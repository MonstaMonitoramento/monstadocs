Além dos links para cada tela, há recursos disponíveis na barra superior do Monsta.

## Informações

![image-1739970120731.png](../../../../../assets/images/p3_image-1739970120731.png)

No canto superior direito o Monsta exibe informações sobre os recursos disponíveis no sistema.

| Ícone | Descrição |
| :---: | :--- |
| ![image-1739970141784.png](../../../../../assets/images/p3_image-1739970141784.png) | SMS: Este campo informa a quantidade disponível de SMS para serem enviados em caso de mudança de status de algum dispositivo ou monitor. |
| ![image-1739970160223.png](../../../../../assets/images/p3_image-1739970160223.png) | E-mail: Este campo informa a quantidade disponível de e-mails para serem enviados em caso de mudança de status de algum dispositivo ou monitor. |
| ![image-1732551964229.png](../../../../../assets/images/p3_image-1732551964229.png) | Central de Avisos: Informa quando existem mensagens sobre o Monsta. |
| ![image-1732551901915.png](../../../../../assets/images/p3_image-1732551901915.png) | Comunicação com a Nuvem: Este campo informa se o Monsta está se comunicando com a Nuvem em https://mind.monsta.com.br. Essa comunicação serve para o envio de SMS's e E-mails de alerta, para o backup automático das configurações e para restaurar o backup. |

## Ferramentas

![image-1732552006425.png](../../../../../assets/images/p3_image-1732552006425.png)
Ferramentas: Permite o acesso rápido à diversas ferramentas.

![image-1739970189090.png](../../../../../assets/images/p3_image-1739970189090.png) 
- **Modo noturno**:  Recurso que inverte as cores das telas do Monsta.  
- **Busca global**:  Efetua uma busca pelas palavras solicitadas em toda a estrutura de dados do Monsta.
- **Alterar a Senha**: Permite alterar a senha do usuário atual.
- **Licença e conta**: Retorna informações sobre a licença utilizada.
- **Atualizações**: Permite atualizar o Monsta quando houver uma nova versão.
- **Ajuda**: Leva à página da Wiki do Monsta.
- **Sobre o Monsta**: Retorna informações sobre a versão atual do software. Nesta opção também é possível alterar o usuário no qual sua licença pertence.
- **Sair**: Efetua o logoff do sistema.
