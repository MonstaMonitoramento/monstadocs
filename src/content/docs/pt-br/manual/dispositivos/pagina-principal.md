Nesta tela são exibidos todos dispositivos monitorados na rede e seus respectivos status. Aqui é possível criar novos dispositivos e adicionar monitores para cada um deles, assim como visualizar os gráficos das coletas efetuadas pelos monitores selecionados e personalizá-los.

![image-1739970299845.png](../../../../../assets/images/p52_image-1739970299845.png)

## Modos de Visualização

| Ícone | Descrição |
| :---: | :--- |
| ![image-1646846764087.png](../../../../../assets/images/p52_image-1646846764087.png) | **Grade**: Visualiza os dispositivos em formato de caixa. É a tela padrão de entrada.<br />Para maiores informações, consulte [Visualização em Modo Grid](/pt-br/manual/dispositivos/visualizacao-em-modo-grid). |
| ![image-1646846862499.png](../../../../../assets/images/p52_image-1646846862499.png) | **Visão detalhada**: Traz uma visão macro sobre um dispositivo que foi selecionado. |
| ![image-1646846951852.png](../../../../../assets/images/p52_image-1646846951852.png) | **Visão de grupos**: Mostra os grupos de dispositivos existente. Ao clicar sobre um grupo, um filtro é ativado e serão mostrados apenas os dispositivos referente ao grupo selecionado. |
| ![image-1646847134431.png](../../../../../assets/images/p52_image-1646847134431.png) | **Visão sunburst**: A partir do dispositivo pai selecionado essa visão apresenta uma estrutura da rede com os equipamentos organizados de forma hierárquica. Também é apresentado um resumo dos status e mudanças na linha do tempo.<br />Para maiores informações, consulte [Visualização Sunburst](/pt-br/manual/dispositivos/visualizacao-sunburst). |
| ![image-1739970604314.png](../../../../../assets/images/p52_image-1739970604314.png) | **Visão de mapa**: Mostra uma visão gráfica e hierárquica dos dispositivos.<br />Para maiores informações, consulte [Visualização em Mapa](/pt-br/manual/dispositivos/visualizacao-em-mapa). |

## Propriedades do Dispositivo

![image-1746721902542.png](../../../../../assets/images/p52_image-1746721902542.png)

| Ícone | Descrição |
| :---: | :--- |
| ![image-1746721988339.png](../../../../../assets/images/p52_image-1746721988339.png) | **Ativa Monitoramento**:Esta chave, quando inativa, pausa a coleta de informações contra o dispositivo. |
| ![image-1746722104491.png](../../../../../assets/images/p52_image-1746722104491.png) | **Visão Detalhada**: Abra uma nova janela com várias informações sobre o dispositivo, como uptime, status dos monitores e sua linha do tempo. |
| ![image-1746722237020.png](../../../../../assets/images/p52_image-1746722237020.png) | **Linha do Tempo**: Abre a [Linha do Tempo](/pt-br/manual/linha-tempo/linha-do-tempo) com o filtro ativado para esse dispositivo. |
| ![image-1746722341442.png](../../../../../assets/images/p52_image-1746722341442.png) | **Editar**: Abre a edição do dispositivo. Consulte [Novo Dispositivo](/pt-br/manual/dispositivos/novo-dispositivo) para maiores informações. |
| ![image-1746722438462.png](../../../../../assets/images/p52_image-1746722438462.png) | **Excluir**: Deleta o dispositivo selecionado e todos seus monitores. |
| ![image-1746722493757.png](../../../../../assets/images/p52_image-1746722493757.png) | **Clonar Dispositivo**: Cria uma cópia do dispositivo selecionado juntamente com todos seus monitores. |
| ![image-1746722576309.png](../../../../../assets/images/p52_image-1746722576309.png) | **Monitores Automáticos**: Esta opção cria regras para que o Monsta adicione monitores de forma automática ao dispositivo selecionado. Para maiores informações, consulte " |[Monitores Automáticos](https://wiki.monsta.com.br/books/manual-do-usuario/page/monitores-automaticos)".
| ![image-1746722650939.png](../../../../../assets/images/p52_image-1746722650939.png) | **Adicionar Monitores**: Adicionar monitores do template selecionado para o dispositivo ou permite a criação de um monitor personalizado apenas para ele. |
