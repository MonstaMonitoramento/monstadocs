Gerencie a biblioteca de sons para seus alertas: faça upload e exclua o que não precisa mais.

![image-1739990942148.png](../../../../../assets/images/p75_image-1739990942148.png)

| Ícone / Opção | Descrição |

| Nome | Nome do arquivo de som. |
| Categoria | Define uma categoria ao som selecionado. |
| ![image-1739991035362.png](../../../../../assets/images/p75_image-1739991035362.png) | Executa o arquivo de som selecionado. |
| ![image-1739991057487.png](../../../../../assets/images/p75_image-1739991057487.png) | Informa o identificador da imagem selecionada. |
| ![image-1739991080033.png](../../../../../assets/images/p75_image-1739991080033.png) | Efetua o download de um arquivo. |
| ![image-1739990449233.png](../../../../../assets/images/p74_image-1739990449233.png) | Remove a imagem selecionada |
