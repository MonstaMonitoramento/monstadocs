Neste local é possível personalizar todos dados relacionados a configuração do Monsta e também a restauração de backups do banco de dados.

![image-1756128994828.png](../../../../../assets/images/p60_image-1756128994828.png)


| Opção | Descrição |
| :---: | :--- |
| ![image-1647434370176.png](../../../../../assets/images/p60_image-1647434370176.png) | Quando ativado, inverte as cores de tela do Monsta. |
| ![image-1647435194179.png](../../../../../assets/images/p60_image-1647435194179.png) | Permite criar, editar e remover templates e monitores existentes. Para mais informações consulte: [Templates](/pt-br/manual/configuracoes/templates). |
| ![image-1647435290317.png](../../../../../assets/images/p60_image-1647435290317.png) | Permite personalizar configurações dos agentes e bloqueá-los. Para mais informações consulte: [Agentes](/pt-br/manual/configuracoes/agentes). |
| ![image-1647435557586.png](../../../../../assets/images/p60_image-1647435557586.png) | Permite adicionar e remover grupos de dispositivos bem como gerenciá-los. Para mais informações consulte: [Grupo de Dispositivos](/pt-br/manual/configuracoes/grupo-de-dispositivos). |
| ![image-1647435611989.png](../../../../../assets/images/p60_image-1647435611989.png) | Permite criar, editar e remover as fontes de dados, buscas de instâncias, métodos de uptime e ações. Para mais informações consulte: [Monsta Studio](/pt-br/manual/configuracoes/monsta-studio). |
| ![image-1647435653352.png](../../../../../assets/images/p60_image-1647435653352.png) | Gerencia a biblioteca de imagens do Monsta. Para mais informações consulte: [Imagens](/pt-br/manual/configuracoes/imagens). |
| ![image-1739982775972.png](../../../../../assets/images/p60_image-1739982775972.png) | É a biblioteca de sons do Monsta onde é feito o gerenciamento dos arquivos que estarão disponíveis para os alertas. Para mais informações consulte: [Sons](/pt-br/manual/configuracoes/sons). |
| ![image-1647435745876.png](../../../../../assets/images/p60_image-1647435745876.png) | Alterna entre os canais de atualização do Monsta. Podem ser escolhidos os canais:<br />- **Oficial**: Canal com atualizações consideradas estáveis.<br />- **Beta**: Canal com atualizações em fase de testes.
| ![image-1647436547485.png](../../../../../assets/images/p60_image-1647436547485.png) | Gerencia os usuários no Monsta. Para mais informações consulte: [Usuários](/pt-br/manual/configuracoes/usuarios). |
| ![image-1647436903151.png](../../../../../assets/images/p60_image-1647436903151.png) | Permite alterar os parâmetros de configuração geral do Monsta.<br /><aside class="starlight-aside starlight-aside--danger"><p class="starlight-aside__title">Cuidado</p>Parâmetros configurados de forma incorreta podem tornar o Monsta inoperante.</aside><br />Para mais informações consulte: [Parâmetros](/pt-br/manual/configuracoes/parametros). |
| ![image-1647437280055.png](../../../../../assets/images/p60_image-1647437280055.png) | Permite ativar ou desativar recursos disponíveis no Monsta. Para mais informações consulte: [Plugins](/pt-br/manual/configuracoes/plugins). |
| ![image-1756129051901.png](../../../../../assets/images/p60_image-1756129051901.png) | Nessa opção é possível visualizar o status dos serviços do Monsta e, se necessário, reiniciá-los individualmente. Para mais informações consulte: [Serviços](/pt-br/manual/configuracoes/servicos). |
| ![image-1647435477277.png](../../../../../assets/images/p60_image-1647435477277.png) | Nesta opção é possível restaurar um backup das configurações do Monsta. Para mais informações consulte: [Backup em Nuvem](/pt-br/manual/configuracoes/backup-em-nuvem). |
| ![image-1739983055127.png](../../../../../assets/images/p60_image-1739983055127.png) | Instale e gerencie seus certificados para acessar a interface web do Monsta. É possível utilizar certificados próprios, gerados pelo Monsta ou automatizá-los através do Letsencrypt. Para mais informações consulte: [Certificado TLS](/pt-br/manual/configuracoes/certificado-tls). |
| ![image-1647437341171.png](../../../../../assets/images/p60_image-1647437341171.png) | Apresenta informações sobre a chave de licença utilizada. Para mais informações consulte: [Chave de Licença](/pt-br/manual/configuracoes/chave-de-licenca). |
| ![image-1647437392464.png](../../../../../assets/images/p60_image-1647437392464.png) | Apresenta informações sobre a versão atual. |
