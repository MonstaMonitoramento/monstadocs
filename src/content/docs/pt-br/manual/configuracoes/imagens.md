Gerencie suas imagens com facilidade: faça upload, organize por grupos e nomes personalizados, e exclua o que não precisa mais. Sua biblioteca de imagens, do seu jeito.

![image-1739990239560.png](../../../../../assets/images/p74_image-1739990239560.png)

| Ícone / Opção | Descrição |
| :---: | :--- |
| Nome | Define um nome para a imagem. |
| Grupo | Define a qual grupo a imagem pertence. |
| ![image-1739990383549.png](../../../../../assets/images/p74_image-1739990383549.png) | Informa o identificador da imagem selecionada. |
| ![image-1739990449233.png](../../../../../assets/images/p74_image-1739990449233.png) | Remove a imagem selecionada |
| ![image-1739990471015.png](../../../../../assets/images/p74_image-1739990471015.png) | Salva a imagem selecionada e seus atributos. |
| ![image-1739990495811.png](../../../../../assets/images/p74_image-1739990495811.png) | Efetua o upload de um arquivo para o banco de imagens. |
| ![image-1739990524318.png](../../../../../assets/images/p74_image-1739990524318.png) | Filtra aa lista de imagens conforme o texto informado. |
| ![image-1739990595935.png](../../../../../assets/images/p74_image-1739990595935.png) | Exibe apenas as imagens pertencentes ao grupo selecionado. |
