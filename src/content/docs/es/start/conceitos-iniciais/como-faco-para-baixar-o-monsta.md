---
title: "¿Cómo puedo descargar Monsta?"
sidebar:
  order: 5
---

Para descargar Monsta, siga estos pasos:

- **Acceda al sitio de descargas**: Vaya a [https://monsta.com.br/downloads](https://monsta.com.br/downloads).
- **Elija el sistema operativo**: Monsta puede descargarse para distribuciones Linux que utilizan paquetes .RPM, como Fedora y Red Hat, así como para aquellas que utilizan .DEB, como Debian y Ubuntu.
- **Siga las instrucciones de instalación**: Acceda al [Manual de instalación](/es/start/instalacao/instalacao-monsta#instalación) para más información sobre cómo instalar Monsta en su servidor.
