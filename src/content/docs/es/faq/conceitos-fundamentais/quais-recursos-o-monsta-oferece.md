---
title: "¿Qué características ofrece Monsta?"
---

Monsta ofrece un conjunto sólido de funcionalidades para la monitorización integral de la infraestructura de TI. Estas funcionalidades están diseñadas para proporcionar visibilidad, detección proactiva de problemas, alertas eficaces y herramientas de análisis para garantizar la salud y el rendimiento de sus sistemas y servicios.

## Recursos principales de Monsta

### 1. Monitorización en tiempo real

- **Monitorización de servidores**: Seguimiento de métricas como CPU, memoria RAM, disco (espacio libre, IOPS), uso de red, procesos y servicios en ejecución (Windows, Linux).
- **Monitorización de red**: Seguimiento de dispositivos de red (routers, switches, firewalls) vía SNMP, ICMP y otros protocolos, monitorizando tráfico, uso de interfaces, errores, estado y tablas de enrutamiento.
- **Monitorización de aplicaciones y servicios**: Monitorización de aplicaciones web (HTTP, HTTPS), servidores de correo (SMTP, POP3, IMAP), DNS y otros servicios críticos, verificando disponibilidad y rendimiento.
- **Monitorización de entornos virtuales**: Soporte para el seguimiento de plataformas de virtualización como VMware y Hyper-V, monitorizando el rendimiento de hosts y máquinas virtuales.

### 2. Visualización y paneles

- **Paneles personalizables**: Creación de paneles gráficos intuitivos y dinámicos para mostrar las métricas más importantes según las necesidades de cada usuario o equipo.
- **Gráficos en tiempo real**: Visualización gráfica del rendimiento de los recursos a lo largo del tiempo, facilitando la identificación de tendencias y anomalías.
- **Mapas de infraestructura**: Representación visual de la topología de la red y de las relaciones entre los dispositivos y servicios monitorizados.
- **Vistas personalizadas**: Creación de vistas específicas para diferentes áreas de la infraestructura o para usuarios con distintos niveles de acceso.

### 3. Alertas y notificaciones

- **Configuración de umbrales**: Definición de límites para las métricas monitorizadas, generando alertas cuando dichos límites se superan.
- **Notificaciones**: Envío de alertas por correo electrónico, SMS y Telegram.

### 4. Análisis e informes

- **Historial de datos**: Almacenamiento de datos de monitorización para el análisis de tendencias a largo plazo y con fines de informes.
- **Análisis de causa raíz**: Herramientas para ayudar a identificar la causa raíz de problemas de rendimiento o fallos.
- **Planificación de capacidad**: Los datos históricos pueden usarse para prever necesidades futuras de capacidad.

### 5. Otros recursos importantes

- **Copia de seguridad automática en la nube**: Seguridad de los datos de configuración y monitorización.
- **Gestión de usuarios y permisos**: Control de acceso a la plataforma con diferentes niveles de permiso.
- **API (Interfaz de Programación de Aplicaciones)**: Posibilidad de integrar Monsta con otras herramientas y sistemas.
- **Agentes y monitorización remota**: Flexibilidad para monitorizar recursos en diferentes ubicaciones.
- **Facilidad de uso**: Interfaz intuitiva y configuración simplificada.
