Además de los enlaces a cada pantalla, hay recursos disponibles en la barra superior de Monsta.

## Información

![image-1739970120731.png](../../../../../assets/images/p3_image-1739970120731.png)

En la esquina superior derecha Monsta muestra información sobre los recursos disponibles en el sistema.

| Icono | Descripción |
| :---: | :--- |
| ![image-1739970141784.png](../../../../../assets/images/p3_image-1739970141784.png) | SMS: Este campo informa la cantidad disponible de SMS para ser enviados en caso de cambio de estado de algún dispositivo o monitor. |
| ![image-1739970160223.png](../../../../../assets/images/p3_image-1739970160223.png) | E-mail: Este campo informa la cantidad disponible de correos electrónicos para ser enviados en caso de cambio de estado de algún dispositivo o monitor. |
| ![image-1732551964229.png](../../../../../assets/images/p3_image-1732551964229.png) | Centro de Avisos: Informa cuando hay mensajes sobre Monsta. |
| ![image-1732551901915.png](../../../../../assets/images/p3_image-1732551901915.png) | Comunicación con la Nube: Este campo indica si Monsta está comunicándose con la Nube en https://mind.monsta.com.br. Esta comunicación sirve para el envío de SMS y correos electrónicos de alerta, para la copia de seguridad automática de las configuraciones y para restaurar la copia de seguridad. |

## Herramientas

![image-1732552006425.png](../../../../../assets/images/p3_image-1732552006425.png)
Herramientas: Permite el acceso rápido a diversas herramientas.

![image-1739970189090.png](../../../../../assets/images/p3_image-1739970189090.png) 
- **Modo nocturno**: Recurso que invierte los colores de las pantallas de Monsta.  
- **Búsqueda global**: Realiza una búsqueda de las palabras solicitadas en toda la estructura de datos de Monsta.
- **Cambiar la contraseña**: Permite cambiar la contraseña del usuario actual.
- **Licencia y cuenta**: Devuelve información sobre la licencia utilizada.
- **Actualizaciones**: Permite actualizar Monsta cuando haya una nueva versión.
- **Ayuda**: Lleva a la página de la Wiki de Monsta.
- **Acerca de Monsta**: Devuelve información sobre la versión actual del software. En esta opción también es posible cambiar el usuario al que pertenece su licencia.
- **Salir**: Cierra la sesión del sistema.
