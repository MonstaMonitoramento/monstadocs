En esta sección es posible personalizar todos los datos relacionados con la configuración de Monsta y también la restauración de copias de seguridad de la base de datos.

| ![image-1647435194179.png](../../../../../assets/images/p60_image-1647435194179.png) | Permite crear, editar y eliminar templates y monitores existentes. Para más información consulte: [Plantillas](/es/manual/configuracoes/templates). |
| ![image-1647435290317.png](../../../../../assets/images/p60_image-1647435290317.png) | Permite personalizar la configuración de los agentes y bloquearlos. Para más información consulte: [Agentes](/es/manual/configuracoes/agentes). |
| ![image-1647435557586.png](../../../../../assets/images/p60_image-1647435557586.png) | Permite añadir y eliminar grupos de dispositivos así como gestionarlos. Para más información consulte: [Grupo de Dispositivos](/es/manual/configuracoes/grupo-de-dispositivos). |
| ![image-1647435611989.png](../../../../../assets/images/p60_image-1647435611989.png) | Permite crear, editar y eliminar fuentes de datos, búsquedas de instancias, métodos de uptime y acciones. Para más información consulte: [Monsta Studio](/es/manual/configuracoes/monsta-studio). |
| ![image-1647435653352.png](../../../../../assets/images/p60_image-1647435653352.png) | Gestiona la biblioteca de imágenes de Monsta. Para más información consulte: [Imágenes](/es/manual/configuracoes/imagens). |
| ![image-1739982775972.png](../../../../../assets/images/p60_image-1739982775972.png) | Es la biblioteca de sonidos de Monsta donde se gestionan los archivos que estarán disponibles para las alertas. Para más información consulte: [Sonidos](/es/manual/configuracoes/sons). |
| ![image-1647435745876.png](../../../../../assets/images/p60_image-1647435745876.png) | Alterna entre los canales de actualización de Monsta. | ![image-1647436547485.png](../../../../../assets/images/p60_image-1647436547485.png) | Gestiona los usuarios en Monsta. Para más información consulte: [Usuarios](/es/manual/configuracoes/usuarios). |
| ![image-1647436903151.png](../../../../../assets/images/p60_image-1647436903151.png) | Permite cambiar los parámetros de configuración general de Monsta.<br /><aside class="starlight-aside starlight-aside--danger"><p class="starlight-aside__title">Precaución</p>Los parámetros configurados de forma incorrecta pueden dejar a Monsta inoperativo.</aside><br />Para más información consulte: [Parámetros](/es/manual/configuracoes/parametros). |
| ![image-1647437280055.png](../../../../../assets/images/p60_image-1647437280055.png) | Permite activar o desactivar recursos disponibles en Monsta. Para más información consulte: [Complementos](/es/manual/configuracoes/plugins). |
