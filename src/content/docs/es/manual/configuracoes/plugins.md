En la pantalla de complementos, verá una lista con todos los plugins disponibles para cada tipo de suscripción. Cada plugin va acompañado de una breve descripción de su funcionalidad y de un botón para activarlo o desactivarlo.



:::danger[Atención]
Desactivar plugins hará que algunos recursos no estén disponibles. Asegúrese de no utilizarlos.
:::
