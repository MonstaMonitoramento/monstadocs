Gestione sus imágenes con facilidad: suba archivos, organícelas por grupos y nombres personalizados, y elimine lo que ya no necesite. Su biblioteca de imágenes, a su manera.

![image-1739990239560.png](../../../../../assets/images/p74_image-1739990239560.png)

| Ícone / Opção | Descrição |
| :---: | :--- |
| Nome | Define un nombre para la imagen. |
| Grupo | Define a qué grupo pertenece la imagen. |
| ![image-1739990383549.png](../../../../../assets/images/p74_image-1739990383549.png) | Muestra el identificador de la imagen seleccionada. |
| ![image-1739990449233.png](../../../../../assets/images/p74_image-1739990449233.png) | Elimina la imagen seleccionada |
| ![image-1739990471015.png](../../../../../assets/images/p74_image-1739990471015.png) | Guarda la imagen seleccionada y sus atributos. |
| ![image-1739990495811.png](../../../../../assets/images/p74_image-1739990495811.png) | Sube un archivo a la biblioteca de imágenes. |
| ![image-1739990524318.png](../../../../../assets/images/p74_image-1739990524318.png) | Filtra la lista de imágenes según el texto informado. |
| ![image-1739990595935.png](../../../../../assets/images/p74_image-1739990595935.png) | Muestra solo las imágenes pertenecientes al grupo seleccionado. |
