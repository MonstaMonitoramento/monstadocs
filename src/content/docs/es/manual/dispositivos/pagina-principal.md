En esta pantalla se muestran todos los dispositivos monitorizados en la red y sus respectivos estados. Aquí es posible crear nuevos dispositivos y añadir monitores para cada uno de ellos, así como visualizar los gráficos de las recopilaciones realizadas por los monitores seleccionados y personalizarlos.

![image-1739970299845.png](../../../../../assets/images/p52_image-1739970299845.png)

## Modos de visualización

| Ícone | Descrição |
| :---: | :--- |
| ![image-1646846764087.png](../../../../../assets/images/p52_image-1646846764087.png) | **Cuadrícula**: Visualiza los dispositivos en formato de caja. Es la pantalla predeterminada de entrada.<br />Para más información, consulte [Visualización en modo Grid](/es/manual/dispositivos/visualizacao-em-modo-grid). |
| ![image-1646846862499.png](../../../../../assets/images/p52_image-1646846862499.png) | **Vista detallada**: Ofrece una visión general del dispositivo seleccionado. |
| ![image-1646846951852.png](../../../../../assets/images/p52_image-1646846951852.png) | **Vista de grupos**: Muestra los grupos de dispositivos existentes. Al hacer clic en un grupo, se activa un filtro y se mostrarán solo los dispositivos pertenecientes al grupo seleccionado. |
| ![image-1646847134431.png](../../../../../assets/images/p52_image-1646847134431.png) | **Vista sunburst**: A partir del dispositivo padre seleccionado, esta vista presenta una estructura de la red con los equipos organizados de forma jerárquica. También se muestra un resumen de los estados y los cambios en la línea de tiempo.<br />Para más información, consulte [Visualización Sunburst](/es/manual/dispositivos/visualizacao-sunburst). |
| ![image-1739970604314.png](../../../../../assets/images/p52_image-1739970604314.png) | **Vista de mapa**: Muestra una vista gráfica y jerárquica de los dispositivos.<br />Para más información, consulte [Visualización en Mapa](/es/manual/dispositivos/visualizacao-em-mapa). |

## Propiedades del dispositivo

![image-1746721902542.png](../../../../../assets/images/p52_image-1746721902542.png)

| Ícone | Descrição |
| :---: | :--- |
| ![image-1746721988339.png](../../../../../assets/images/p52_image-1746721988339.png) | **Activa monitorización**: Esta opción, cuando está inactiva, pausa la recopilación de información sobre el dispositivo. |
| ![image-1746722104491.png](../../../../../assets/images/p52_image-1746722104491.png) | **Vista detallada**: Abre una nueva ventana con varias informaciones sobre el dispositivo, como el tiempo de actividad (uptime), el estado de los monitores y su línea de tiempo. |
| ![image-1746722237020.png](../../../../../assets/images/p52_image-1746722237020.png) | **Línea de tiempo**: Abre la [Línea de Tiempo](/es/manual/linha-tempo/linha-do-tempo) con el filtro activado para este dispositivo. |
| ![image-1746722341442.png](../../../../../assets/images/p52_image-1746722341442.png) | **Editar**: Abre la edición del dispositivo. Consulte [Nuevo Dispositivo](/es/manual/dispositivos/novo-dispositivo) para más información. |
| ![image-1746722438462.png](../../../../../assets/images/p52_image-1746722438462.png) | **Eliminar**: Elimina el dispositivo seleccionado y todos sus monitores. |
| ![image-1746722493757.png](../../../../../assets/images/p52_image-1746722493757.png) | **Clonar dispositivo**: Crea una copia del dispositivo seleccionado junto con todos sus monitores. |
| ![image-1746722576309.png](../../../../../assets/images/p52_image-1746722576309.png) | **Monitores automáticos**: Esta opción crea reglas para que el Monsta añada monitores de forma automática al dispositivo seleccionado. Para más información, consulte " |[Monitores Automáticos](https://wiki.monsta.com.br/books/manual-do-usuario/page/monitores-automaticos)".
| ![image-1746722650939.png](../../../../../assets/images/p52_image-1746722650939.png) | **Añadir monitores**: Añade monitores de la plantilla seleccionada al dispositivo o permite la creación de un monitor personalizado únicamente para él. |
