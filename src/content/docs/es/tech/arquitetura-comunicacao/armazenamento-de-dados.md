Todos los datos de monitorización recopilados de sus dispositivos y monitores se almacenan en una estructura de base de datos diseñada para garantizar velocidad, integridad y precisión histórica.

### Qué significa esto para usted



| **Característica Técnica** | **Beneficio para el Usuario** |
| --- | --- |
| **NoSQL (No Relacional)** | Garantiza **alta velocidad** y **eficiencia** en el almacenamiento y recuperación de grandes volúmenes de datos (Big Data), permitiendo que el sistema siga las métricas de innumerables dispositivos en tiempo real. |
| **Serie Temporal** | Los datos se almacenan con una **marca de tiempo exacta**, organizados en orden cronológico. Esto optimiza el acceso y el análisis de **tendencias y patrones a lo largo del tiempo**, siendo ideal para la monitorización. |
| **No se pueden modificar/eliminar/restablecer** | Los datos recopilados de los monitores no pueden manipularse por motivos de seguridad. Una vez que un dato (evento, métrica) se registra, se vuelve **permanente**. Esta característica es importante para **auditoría, cumplimiento y forense de TI**, ya que garantiza que el historial nunca ha sido manipulado. |



En resumen, la combinación de estas tecnologías garantiza que usted tenga acceso a un **historial de rendimiento rápido, fiable y a prueba de manipulación**.
